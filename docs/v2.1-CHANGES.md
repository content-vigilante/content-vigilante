# v2.1 — what's new

Headline: **Context library**. Drop research / transcripts / briefs into the app, get strategic content ideas back.

## Context library (new)
- New `/dashboard/context` page with drag-and-drop file ingestion.
- Supported in v2.1: PDF, Markdown, plain text, CSV/TSV, JSON, HTML, SRT/VTT subtitle files.
- All extracted text lives in the browser (`localStorage` → `cv:store:context`). The cross-device sync layer picks it up automatically when KV is configured.
- Tag and search across items. Filter by tag.
- New `/api/context/extract` endpoint — server-side text extraction.
- New `/api/suggest` endpoint — sends your context + brand rules + goal to the LLM and returns 5 structured idea objects (title, angle, ready-to-edit draft, source attribution).
- "Open in Studio" hands the draft over to the composer with the platform pre-selected.

## Polish / pre-existing v1.3 items folded in
- **Mobile responsive sidebar** — collapses to a hamburger drawer below `md`.
- **Brand-aware OG image** at `/opengraph-image` (Fraunces + cream/navy/green).
- **Favicon** (`/icon.svg`) using the owl mark.
- **Global error boundary** with branded fallback + 404 page.
- **Auto-sync** — debounced 4s push to `/api/sync` after any tracked-key mutation. Toggle in Settings.
- **Studio enhancements**:
  - Headline scorecard (length, power words, numbers, question form, CTR predictors)
  - SEO density (target keyword + density + advice)
  - Bias & sensitivity detector (gendered, ableist, loaded technical metaphors, tech-bro clichés)
  - Smart emoji suggester (sentiment-driven, not random)
- **Lead scoring** in Pipeline — stage weight + value + source quality + recency → 0–100 tier (hot / warm / cold).
- **Time & effort tracker** — floating widget; creation vs engagement; today's total; entries log.
- **Inbox** — unified comment aggregator across channels, with one-click "to CRM" conversion.
- **Exports**:
  - Portfolio summary (campaigns / published / scheduled / closed revenue)
  - Freelance invoice generator (print-to-PDF ready)
  - Client approval portal (shareable read-only link via sync token)
- **Cron + scheduled publish (browser-driven)** — Calendar "Publish N due" action publishes posts whose `scheduledFor` is in the past via the existing LinkedIn / X endpoints.
